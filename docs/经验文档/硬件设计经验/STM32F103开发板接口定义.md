| # | 模块 | 外设说明 | 使用引脚 | 备注 |

| 1 | 数码管显示 | 74CH245 段选 + 74HC138 位选（2个四位共8位） | PA0～PA7, PB3～PB5 | ⚠️ 与 LED 矩阵、LED 模块冲突 |
| 2 | 串口通讯 | CH340C USB 转串口 | PA9, PA10 | ✅ 无冲突 |
| 3 | 独立按键 | 4 个独立按键 | PA12～PA15 | ⚠️ PA13/PA14 为 SWD 调试口 |

| 6 | OLED 显示 | 0.96寸 I2C OLED | PB6, PB7 | ⚠️ 与 EEPROM 共用 I2C 总线 |
| 7 | EEPROM | I2C EEPROM | PB6, PB7 | ⚠️ 与 OLED 共用 I2C 总线 |
| 8 | ADC 采集 | 可调电阻 | PB1 | ✅ 无冲突 |
| 9 | LED 矩阵 | 74HC595D 行控 + 直控列控（8×8） | PA0～PA7, PB3～PB5 | ⚠️ 三方共用引脚 |
| 10 | LED 模块 | 8 个独立 LED（低电平有效） | PA0～PA7 | ⚠️ 与数码管、LED 矩阵冲突 |
| 11 | 电机驱动 | ULN2003D | PB0, PB8, PB9, PB12, PB13 | ⚠️ 多处冲突 |
| 12 | 红外收发 | IRED | PA8 | ✅ 无冲突 |
| 13 | 温度采集 | DS18B20 | PA11 | ✅ 无冲突 |
| 14 | ESP8266 WiFi | ESP8266 模块 | PB2, PB10, PB11, PA15 | ⚠️ 多处冲突 |

## 📝 各模块引脚详情

### 1. 数码管显示

| STM32 引脚 | 信号名 | 说明 | 冲突引脚 |
|-----------|--------|------|---------|
| PA0 | A0 | 段选 a | LED1(PA0), COL1(PA0) |
| PA1 | A1 | 段选 b | LED2(PA1), COL2(PA1) |
| PA2 | A2 | 段选 c | LED3(PA2), COL3(PA2) |
| PA3 | A3 | 段选 d | LED4(PA3), COL4(PA3) |
| PA4 | A4 | 段选 e | LED5(PA4), COL5(PA4) |
| PA5 | A5 | 段选 f | LED6(PA5), COL6(PA5) |
| PA6 | A6 | 段选 g | LED7(PA6), COL7(PA6) |
| PA7 | A7 | 段选 dp | LED8(PA7), COL8(PA7) |
| PB5 | A | 位选 A | RCLK(PB5) |
| PB4 | B | 位选 B | SER(PB4) |
| PB3 | C | 位选 C | SRCLK(PB3) |

**可实现项目：**

- 🔢 **数字时钟** — 动态扫描显示时分秒，配合按键校时
- 📊 **电压表/电流表** — 显示 ADC 采集的模拟量数值
- 🌡️ **温度显示** — 读取 DS18B20 后显示当前温度
- ⏱️ **倒计时/秒表** — 按键启动/停止，数码管显示计时
- 🎮 **猜数字游戏** — 随机生成数字，按键猜大小
- 📡 **串口数据接收显示** — 上位机发送数字，数码管实时显示

## 💡 推荐组合方案

> 以下方案在引脚互斥约束下可同时工作：

### 方案 A：数据采集与显示站

**模块组合：** 串口通讯 + OLED 显示 + EEPROM + ADC 采集 + DS18B20 + 蜂鸣器

> 温度+电压采集 → OLED 显示 → EEPROM 存储 → 超限报警

---

### 方案 B：交互式终端

**模块组合：** 串口通讯 + OLED 显示 + 独立按键 + 蜂鸣器 + ADC

> 菜单系统 + 参数调节 + 按键操作 + 提示音

---

### 方案 C：网络监控节点

**模块组合：** OLED 显示 + ESP8266 WiFi + DS18B20 + 蜂鸣器

> 温度采集 → WiFi 上传 → OLED 显示 → 超限报警

---

### 方案 D：游戏终端

**模块组合：** OLED 显示 + 矩阵按键 + 蜂鸣器

> 俄罗斯方块/贪吃蛇 + 键盘控制 + 音效

---

### 方案 E：工业控制面板

**模块组合：** 数码管显示 + 矩阵按键 + 电机驱动 + 蜂鸣器 + LED 模块

> 数码管显示参数 + 键盘输入 + 电机执行 + 状态指示
